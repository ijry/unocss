社区预设 ​

  • unocss-preset-scalpel - 由@macheteHot制作的 Scalpel 预设。
  • unocss-preset-chroma - 由@chu121su12制作的 Gradient 预设。
  • unocss-preset-scrollbar - 由@action-hong制作的 Scrollbar 预设。
  • unocss-applet - 由@zguolee制作的在小程序（UniApp / Taro）中使用 UnoCSS 的预设。
  • unocss-preset-weapp - 由@MellowCo制作的微信小程序预设，适用于UniApp和Taro。
  • unocss-preset-heropatterns - 由@Julien-R44制作的集成了Hero Patterns的预设。
  • unocss-preset-flowbite - 由@Julien-R44制作的将Flowbite Tailwind 插件移植至 UnoCSS 的预设。
  • unocss-preset-forms - 由@Julien-R44制作的将@tailwindcss/forms移植至 UnoCSS 的预设。
  • unocss-preset-extra - 由@Zhang-Wei-666制作的Animate.css预设以及其他一些规则。
  • unocss-preset-daisy - 由@kidonng制作的 daisyUI 预设。
  • unocss-preset-primitives - 由@zirbest制作的类似于headlessui-tailwindcss、radix-ui的 UnoCSS 定制预设。
  • unocss-preset-theme - 由@Dunqing制作的自动主题切换预设。
  • unocss-preset-chinese - 由@kirklin制作的中文字体预设。
  • unocss-preset-autoprefixer - 由@zouhang制作的 Autoprefixer 预设。
  • unocss-preset-scrollbar-hide - 由@reslear制作的隐藏滚动条预设，尽管元素仍然可以滚动。
  • unocss-preset-radix - 由@endigma制作的使用 Radix UI 调色板的 UnoCSS 预设。
  • unocss-preset-glyph - 由@chu121su12制作的从字体中嵌入子集图标的 UnoCSS 预设。
  • unocss-preset-grid - 由@StatuAgency制作的类似于 Bootstrap 5 的网格预设。
  • unocss-preset-grid-areas - 由@chz制作的 UnoCSS 网格区域预设。
  • unocss-preset-ease - 由@kirklin制作的作为 UnoCSS 过渡的缓动函数预设。
  • unocss-preset-useful - 由@zyyv制作的集成和实用的 UnoCSS 预设。
  • una-ui-preset - 通过@phojie提供的 UnaUI 预设构建自己的组件，使用 UnaUI。
  • unocss-preset-fluid - 由@renatomoor提供的优雅地调整字体和间距而无需断点的 UnoCSS 预设。
  • unocss-preset-block - 由@chizuki制作的阻止一些非标准类名的 UnoCSS 预设。
  • unocss-preset-animations - 由@Aelita提供的用于灵活和可组合动画的 UnoCSS 预设。
  • unocss-preset-shadcn - 由@zouhang和@hyoban提供的使用 shadcn/ui 和 UnoCSS 的预设。
  • @yeungkc/unocss-preset-safe-area - 由YeungKC提供的在 Unocss 中使用安全区域的预设。
  • @unifydev/unify-preset - 由@johnkat-mj提供的由 UnoCSS 提供动力的原子主题 UI 库，类似于 DaisyUI 但可定制化。

社区框架 ​

  • Anu - 由@jd-solanki提供的以开发体验为中心的基于 Vue 的实用组件库。
  • Onu UI - 由@zyyv和@yzh990918提供的倔强而轻量的 UnoCSS UI 库。
  • AtoUI - 由@bennymi提供的为 Svelte 提供的元素 UI 库，提供易于主题化的设计系统。
  • Una UI - 由@phojie提供的用于 Nuxt 3 的原子 UI 框架，由 UnoCSS 引擎驱动。

社区转换器 ​

  • unocss-transformer-attribute-values-group - UnoCSS 属性值分组转换器，作者 @lvjiaxuan。
  • unocss-transformer-alias - UnoCSS 快捷方式的别名转换器，作者 @zyyv。
  • CSS to UnoCss - 将 CSS 转换为 UnoCSS 的工具，作者 @Simon-He95。
  • unocss-transformer-classnames-minifier - UnoCSS 类名缩小转换器，作者 @alisobirjanov。

社区工具 ​

  • Unot - 由@Simon-He95提供的 VSCode 扩展，可以将 CSS 转换为 UnoCSS，并简化 UnoCSS 的编写过程。
  • figma-to-unocss - 由@zouhang提供的 Figma 插件，可以将设计转换为 UnoCSS。
